# ContextMenu 

The `contextMenu` expose two methods: `show` and `hideAll`. 

The `show` method give you more control than the `MenuProvider`. I even recommend that you build your own 
`MenuProvider`.

Just don't forget to call `e.preventDefault` especially when using `onContextMenu` event.

## Show Api
<PropTable of={contextMenu} />

```js
import { contextMenu } from 'react-contexify';
```

## HideAll Api

The `hideAll` method don't take any argument.

```js
contextMenu.hideAll();
```
